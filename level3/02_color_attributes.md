## 目標

- 色の三属性（色相・明度・彩度）をそれぞれ説明できる

- トーンの概念と代表的なトーン名を覚えられる

---

## 1. 色の三属性とは

すべての色は次の **3つの属性（次元）** で表すことができる。

| 属性 | 英語 | 意味 | 範囲 |
|---|---|---|---|
| 色相 | Hue（H） | 色みの種類（赤・青・緑など） | 色相環上の位置 |
| 明度 | Value / Lightness（V / L） | 明るさの度合い | 暗い（0）〜明るい（10） |
| 彩度 | Chroma / Saturation（C / S） | 鮮やかさの度合い | 無彩色（0）〜最大14程度 |

---

## 2. 色相（Hue）

**色相** は、赤・橙・黄・黄緑・緑・青緑・青・青紫・紫・赤紫などの「色みの違い」を表す。

### 色相環

色相を環状に並べたものを **色相環** という。

```
           黄（Y）
        黄緑     橙
      緑           赤橙
    青緑              赤（R）
      青            赤紫
        青紫    紫
           (紫)
```

- **補色（反対色）**: 色相環上で正反対に位置する色同士
  - 例: 赤 ↔ 青緑、黄 ↔ 青紫
- **類似色（隣接色）**: 色相環上で隣り合う色
  - 例: 赤 と 赤橙 と 橙

### PCCSの色相環（24色相）

日本の色彩研究で広く使われる **PCCS** の色相環は **24色相**。
各色に番号（1〜24）が振られている。

| 番号 | 主な色名 | 略号 |
|---|---|---|
| 2 | 赤（Red） | R |
| 8 | 黄（Yellow） | Y |
| 12 | 緑（Green） | G |
| 18 | 青（Blue） | B |
| 22 | 紫（Violet） | V |
| 24 | 赤紫（Red-Purple） | RP |

---

## 3. 明度（Value / Lightness）

**明度** は色の明るさを表す。

- 明度が高い → 白に近い（明るい）
- 明度が低い → 黒に近い（暗い）

### マンセル明度スケール

マンセル表色系では明度を **0〜10** で表す。

```
明度 10  → 完全な白
明度  5  → 中間の灰
明度  0  → 完全な黒
```

### 各色相の最高明度（参考）

| 色 | 純色の明度（目安） |
|---|---|
| 黄（Y） | 約8〜9（最も高い） |
| 赤（R） | 約4 |
| 青（B） | 約3〜4 |
| 紫（V） | 約3 |

!!! tip "ポイント"
    黄色は純色の中で最も明度が高く、紫・青は最も低い。

---

## 4. 彩度（Chroma / Saturation）

**彩度** は色の鮮やかさを表す。

- 彩度が高い → 鮮やかな色（ビビッドカラー）
- 彩度が低い → 灰色に近い、くすんだ色（グレイッシュカラー）
- 彩度 = 0  → 無彩色（白・灰色・黒）

### 彩度の範囲

マンセル表色系では、色相によって最大彩度が異なる（赤は14程度が最大）。
PCCSでは、最大彩度を **9s（サチュレーション9）** として統一的に扱う。

---

## 5. 有彩色と無彩色

### 有彩色（Chromatic Color）
色相を持つすべての色（赤・青・黄など）。
三属性（色相・明度・彩度）のすべてを持つ。

### 無彩色（Achromatic Color）
色相がなく、明度だけで区別される色。
- 白・灰色・黒
- 彩度 = 0

---

## 6. 純色・清色・濁色

### 純色（じゅんしょく）
各色相の中で **最も彩度が高い色**。
- 例: 鮮やかな赤、鮮やかな青

### 清色（せいしょく）
純色に **白または黒のみ** を混ぜた色。

| 種類 | 混ぜるもの | 特徴 |
|---|---|---|
| 明清色（めいせいしょく） | 純色 + 白 | 明るく澄んだ色（パステルカラー） |
| 暗清色（あんせいしょく） | 純色 + 黒 | 暗く深みのある色 |

### 濁色（だくしょく）
純色に **グレーを混ぜた色**、または白と黒を同時に混ぜた色。
- くすんだ、渋みのある色調
- 例: オリーブ色、マスタード色

---

## 7. トーン（Tone）

**トーン** とは、明度と彩度を組み合わせた「色調」の概念。
同じ色相でもトーンが異なれば印象が大きく変わる。

### PCCSの主なトーン

| トーン名 | 略称 | 明度 | 彩度 | 印象 |
|---|---|---|---|---|
| ビビッド（Vivid） | v | 中 | 最高 | 鮮やか・強い・派手 |
| ブライト（Bright） | b | 高め | 高 | 明るい・ハッキリした |
| ストロング（Strong） | s | 中 | 高 | 強い・はっきりした |
| ディープ（Deep） | dp | 低め | 高め | 深い・濃い・充実した |
| ライト（Light） | lt | 高 | 中低 | 明るい・軽い・爽やか |
| ソフト（Soft） | sf | 中 | 中低 | 柔らかい・穏やか |
| ダル（Dull） | d | 中 | 低 | 渋い・くすんだ・地味 |
| ダーク（Dark） | dk | 低 | 中低 | 暗い・重い・硬い |
| ペール（Pale） | p | 最高 | 最低 | 薄い・淡い・軽い |
| ライトグレイッシュ（Light Grayish） | ltg | 高め | 最低 | 明るいグレイッシュ |
| グレイッシュ（Grayish） | g | 中 | 最低 | グレーがかった・落ち着き |
| ダークグレイッシュ（Dark Grayish） | dkg | 低め | 最低 | 暗いグレイッシュ |

> **ポイント**:
> - **ビビッド（v）**: 最も彩度が高いトーン（最も鮮やかな純色に近い）
> - **ペール（p）**: 最も明度が高く彩度が低いトーン（パステルカラー）
> - トーン名は英語の意味と結びつけて覚えると忘れにくい

### トーン配置図（概略）

```
高明度   ペール(p)  ライトグレイッシュ(ltg)  ライト(lt)  ブライト(b)
中明度   グレイッシュ(g)               ソフト(sf)  ストロング(s)  ビビッド(v)
低明度   ダークグレイッシュ(dkg)  ダル(d)  ダーク(dk)  ディープ(dp)
         ←── 低彩度 ─────────────────────────── 高彩度 ──→
```
