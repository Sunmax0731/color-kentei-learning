## 1. 色の対比（Color Contrast）

2色以上の色が隣り合うとき、互いに影響し合って元の色と異なって見える現象。

### 1-1. 明度対比

明度の異なる色が隣り合うと、**明るい色はより明るく、暗い色はより暗く** 見える。

```
例:
同じグレーでも   ┌────┬────┐
               │    │    │
               │ Gy │ Gy │   ← 両方同じグレーだが、
               │    │    │
               └────┴────┘
                白地     黒地
→ 白地のGy は暗く見え、黒地のGy は明るく見える
```

!!! warning "試験頻出"
    同じ色でも背景によって見た目が変わる。

### 1-2. 彩度対比

彩度の異なる色が隣り合うと、**高彩度の色はより鮮やかに、低彩度の色はより鈍く** 見える。

```
例:
同じグレーでも、
・高彩度の色の隣 → グレーがくすんで見える
・低彩度の色の隣 → グレーがはっきり見える
```

### 1-3. 色相対比

異なる色相の色が隣り合うと、**互いに相手の補色方向に色みがずれて** 見える。

```
例:
赤の背景に置いた橙 → 赤の補色（青緑）方向に引っ張られて、黄みがかって見える
```

### 1-4. 補色対比（補色の強調）

**補色関係の2色** が隣り合うと、互いをより鮮やかに引き立て合う。
最も強いコントラストが生まれる。

```
例:
赤と青緑（補色）が隣り合う → 両方ともより鮮やかに見える
```

### 1-5. 縁辺対比（マッハバンド）

明度の異なる2色が隣接するとき、境界付近で**明るい側がより明るく、暗い側がより暗く**見える現象。

```
┌──────────────┬──────────────┐
│   明るい灰   │   暗い灰     │
│              │              │
└──────────────┴──────────────┘
           ↑境界付近で強調される
```

**マッハバンド**: 境界部に見かけ上の縞が現れる現象。

### 1-6. 面積対比

色の面積が変わると印象が変わる。
- 大きな面積 → 明度・彩度が増して見える（より明るく・鮮やかに見える）
- 小さな面積 → 明度・彩度が落ちて見える

!!! example "実用例"
    塗料の小さなサンプルを見て選んだ色を実際に広い壁に塗ると、想像より鮮やかに見える。

## 2. 色の同化（Color Assimilation）

対比とは逆に、隣り合う色が**互いに似て見える**現象。
細かい縞模様や細線が混在するとき起きやすい。

```
例:
白地に赤い細線が引かれていると → 白地全体が赤みがかって見える
黒地に赤い細線が引かれていると → 黒地全体が赤みがかって見える
```

### 同化の起きやすい条件

- 背景色と前景色のパターンが細かい（面積が小さい）
- 色の差が比較的小さい

### 対比との違い

| 現象 | 起きやすい条件 | 結果 |
|---|---|---|
| 対比 | 大きな色面積の差 | 互いに異なって見える（強調） |
| 同化 | 細かいパターン | 互いに似て見える（同化） |

!!! example "実用例"
    ネットに入れた果物は、ネットの色によって果物の色が変わって見える（同化）。

## 3. 色の恒常性（Color Constancy）

照明が変わっても、物体の色が同じに見える傾向。

```
例:
白い紙を昼光（白い光）の下で見ても、夕焼け（橙色の光）の下で見ても、
脳が「白い紙」と認識する
```

- 完全な恒常性ではなく、**部分的**に起きる
- 慣れ（順応）と記憶色が関係する

---

## 4. 記憶色（Memory Color）

**記憶の中で色が実際より鮮やかに・特徴的に覚えられる**傾向。

```
例:
空の青 → 実際より青く鮮やかに記憶される
葉の緑 → 実際より濃い緑として記憶される
```

> **広告・写真への影響**: 食品の写真は記憶色に近い鮮やかな色に調整されることが多い。

---

## 5. 色順応（Chromatic Adaptation）

ある色に満たされた環境に長時間いると、その色に目が慣れて**色みが感じられなくなる**現象。

```
例:
黄色いサングラスをかけて10分経つと → 最初に感じた黄みが薄れてくる
赤い部屋に1時間いると → 赤みが感じにくくなる
```

### 5-1. 明順応・暗順応

| 順応 | 内容 |
|---|---|
| 明順応 | 暗い場所から明るい場所へ → 数秒〜数十秒で適応 |
| 暗順応 | 明るい場所から暗い場所へ → 数分〜30分かけて適応 |

暗順応は**ロドプシン（視物質）の再合成**に時間がかかるため遅い。

---

## 6. プルキンエ現象（Purkinje Effect）

明るいところから暗いところ（薄明視）になると、**赤系の色は暗く（暗くなる）、青系の色は明るく見えやすくなる**現象。

```
昼（明所視）:   赤 > 青 の明るさ
薄暮（薄明視）: 赤 < 青 の明るさ（逆転）
```

**理由**: 明所視では**錐体細胞**（緑〜赤感度が高い）が主役、暗所視では**桿体細胞**（青〜緑感度が高い）が主役。

!!! example "実用例"
    夕暮れ時に赤い花が暗く、青い花が明るく見えやすくなる。

---

## 7. ベツォルト＝ブリュッケ現象

明るさ（輝度）が変わると、**色相の見え方がわずかに変化する**現象。

- 輝度が上がると → 黄・青系が際立つ
- 輝度が下がると → 赤・緑系が際立つ

---

## 8. ハーマングリッド（格子状の錯視）

白い格子の交差点に灰色の点が見える錯視。縁辺対比の一例。
