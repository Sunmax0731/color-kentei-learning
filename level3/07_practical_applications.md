## 1. ファッションと色彩

### 1-1. 服の色が与える印象

ファッションにおける色選びは、**自分をどう見せたいか**に直結する。

| 色 | 与える印象 |
|---|---|
| 白 | 清潔感・清純・若々しい |
| 黒 | 格式・引き締め・高級感 |
| 赤 | 情熱・活動的・目立つ |
| 青・紺 | 誠実・知性・落ち着き |
| 緑 | 自然・清潔・穏やか |
| ベージュ | 上品・落ち着き・自然 |
| ピンク | 可愛さ・女性らしさ・優しさ |

### 1-2. 色の面積効果

- **ボトムス（下半身）に暗い色** → 全体が引き締まって見える（収縮効果）
- **トップス（上半身）に明るい色** → 顔まわりが明るく見える
- **全身同系色** → 縦のラインが強調されてスリムに見える

### 1-3. 配色コーディネート

| 配色 | 印象 |
|---|---|
| 同一色相配色（ワントーン） | 洗練・上品・まとまり |
| 類似色相配色 | 穏やか・ナチュラル |
| 補色配色 | 鮮やか・目立つ・スポーティ |
| 無彩色（白・黒・グレー）でまとめる | モード・シンプル |

## 2. パーソナルカラー

**パーソナルカラー** とは、その人の肌・髪・目の色などの自然な色（ナチュラルカラー）に調和する色のこと。

### 2-1. イエローベース・ブルーベース

| タイプ | 特徴 | 似合う色 |
|---|---|---|
| イエローベース（イエベ） | 肌が黄みがかった暖色系 | コーラルピンク・テラコッタ・カーキなど温かみのある色 |
| ブルーベース（ブルベ） | 肌が青みがかった寒色系 | ローズピンク・ラベンダー・グレーなど青みのある色 |

### 2-2. 4シーズン分類

| シーズン | ベース | 特徴 |
|---|---|---|
| スプリング（春） | イエロー | 明るく華やか・コーラル系 |
| オータム（秋） | イエロー | 深みのある暖色・テラコッタ系 |
| サマー（夏） | ブルー | 柔らかく澄んだ淡色・ラベンダー系 |
| ウィンター（冬） | ブルー | コントラスト強め・ビビッドまたはブラック系 |

> **試験ポイント**: パーソナルカラーは「イエローベースかブルーベース」の区別が基本。4シーズン分類も覚えておく。

## 3. インテリアと色彩

### 3-1. インテリアの色彩計画の基本

インテリアの色は「背景色・主役色・アクセント色」の3層構造で考える。

| 層 | 場所 | 面積比（目安） |
|---|---|---|
| 背景色（ベースカラー） | 壁・天井・床 | 70〜75% |
| 主役色（アソートカラー） | 家具・ファブリック | 20〜25% |
| アクセント色 | 小物・クッション・アート | 5% |

### 3-2. 室内色彩の基本ルール

**天井・壁・床の明度の関係**:
- 天井（高明度・白や明るい色）→ 空間が広く・開放的に見える
- 床（中明度〜低明度）→ 安定感・落ち着き
- 天井を暗い色にすると → 圧迫感・落ち着いた空間

**色の広がり感**:
- 膨張色（白・高明度）を壁に使う → 部屋が広く見える
- 収縮色（暗い色）を壁に使う → 部屋が狭く感じる

### 3-3. 部屋の用途と色彩

| 部屋 | 推奨する色調 | 理由 |
|---|---|---|
| 寝室 | 寒色・低彩度・低明度 | リラックス・安眠効果 |
| 子ども部屋 | 暖色・高彩度・高明度 | 活気・学習意欲 |
| 書斎・勉強部屋 | 青系・中彩度 | 集中力・落ち着き |
| リビング | 暖色・中明度 | くつろぎ・家族のまとまり |
| キッチン・食堂 | 白・クリーム・橙 | 清潔感・食欲増進 |

## 4. 安全色彩とサインシステム

### 4-1. 安全色彩（JIS Z 9101）

色によって**危険・安全情報を直感的に伝える**体系。

| 安全色 | 意味 | 使用例 |
|---|---|---|
| 赤 | 防火・禁止・停止・緊急 | 消火器・STOP信号 |
| 黄赤（橙） | 危険 | 工事現場の標識 |
| 黄 | 注意 | 注意を促す標識・ロープ |
| 緑 | 安全・救護・進行 | 避難口・救急 |
| 青 | 指示・誘導 | 着用義務標識 |
| 赤紫 | 放射能 | 放射線関連施設 |
| 白 | 通路・整理 | 通路表示 |
| 黒 | 補助色（白・黄の補助） | 注意・標識の文字 |

### 4-2. 交通信号の色

国際的に統一されている：
- **赤 = 停止**
- **黄（橙） = 注意・準備**
- **緑 = 進行**

## 5. 環境色彩とユニバーサルデザイン

### 5-1. 環境色彩

建物・街・景観などの環境における色彩計画。

- **修景色彩（しゅうけい）**: 景観に調和するよう建物の色を決める
- **景観ガイドライン**: 地域ごとに建物の色を規制する場合がある

### 5-2. カラーユニバーサルデザイン（CUD）

**色の見え方が異なる人**（色覚異常、高齢者など）でも情報が伝わるようにデザインする考え方。

**色覚異常（色盲・色弱）の種類**:
- **P型（第1色覚異常）**: 赤みの識別が困難
- **D型（第2色覚異常）**: 緑みの識別が困難（最も多い）
- **T型（第3色覚異常）**: 青みの識別が困難（まれ）

**CUDの対策**:
- 色だけで情報を伝えない（形・文字・パターンも使う）
- 赤・緑の組み合わせを避ける、または区別できる工夫をする
- 十分なコントラストを確保する
- 凡例・注記を使う

> **試験頻出**: 「色だけでなく形・文字も活用する」が基本の考え方。

### 5-3. 高齢者の色の見え方

年齢とともに水晶体が黄色みを帯び（老化）、以下の変化が起きる：
- 青・紫系の色が見えにくくなる
- 色のコントラストが低下して区別しにくくなる
- 明暗への順応が遅くなる

## 6. 色彩計画の流れ

```
1. 目的・コンセプトの確認
   ↓
2. 対象（ターゲット）の把握
   ↓
3. 色のイメージスケールで方向性を決定
   ↓
4. 配色の設計（ベース・アソート・アクセント）
   ↓
5. 実施・確認・調整
```
