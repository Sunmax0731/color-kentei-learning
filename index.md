# 色彩検定3級・2級 合格学習プログラム

## 学習概要

| 項目 | 内容 |
|---|---|
| 開始日 | 2026年3月18日（水） |
| **3級試験** | **2026年6月28日（日）** 夏期検定 |
| **2級試験** | **2026年11月8日（日）** 冬期検定 |

## Claude Code との連携ルール

学習の記録・テスト回答はClaude Codeに以下のフォーマットで送信してください。

=== "学習時間の記録"

    ```
    学習開始: W1 光と色の基礎
    （学習中...）
    学習終了: W1 光と色の基礎
    ```

=== "確認テストの回答"

    ```
    テスト回答: 3級 光と色の基礎

    Q1: B
    Q2: 可視光線
    Q3: ...
    ```

=== "模擬試験の受験"

    ```
    模擬試験: 3級 第1回
    ```
