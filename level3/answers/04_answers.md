## 解説

**Q1. 正解: C（赤・緑・青）**
加法混色（光の混色）の三原色は **R（赤）・G（緑）・B（青）**。
減法混色の三原色は C（シアン）・M（マゼンタ）・Y（イエロー）。赤・黄・青は絵具の古典的な「三原色」であって加法混色ではない。

**Q2. 正解: B（黄）**
加法混色 R + G = **Y（黄）**。これは重要な組み合わせなので必ず覚える。
- R + G = Y（黄）
- G + B = C（シアン）
- B + R = M（マゼンタ）

**Q3. 正解: C（黒）**
減法混色（色料）で C + M + Y = **K（黒）**（理論上）。実際の印刷では完全な黒にならないため、CMYK の K（Key Plate = 黒インク）を別途加える。

**Q4. 正解: C（減法混色）**
インクジェットプリンターや印刷は色料（インク）を混ぜるので**減法混色**。画面（テレビ・スマホ）は光の加法混色。

**Q5. 正解: C（緑）**
減法混色 C + Y = **G（緑）**。
- C + M = B（青紫）
- M + Y = R（赤）
- Y + C = G（緑）

**Q6. 正解: D（回転混色）**
コマを高速回転させる混色は**回転混色**（回転体混色）。並置混色は色の点を並べて離れて見る混色（印刷の網点など）。どちらも中間混色の一種。

**Q7. 正解: C（白）**
加法混色で R + G + B の全色を混ぜると**白**になる。混ぜるほど明るくなるのが加法混色の特徴。

**Q8. 正解: B（RGB = 加法混色）**
スマートフォン・テレビの画面は RGB の光の加法混色を使って色を表現している。

**Q9. 正解: C（混ぜた各色の明度の平均値）**
中間混色（回転・並置）では明度は各色の**平均値**になる。加法混色のように明るくなるわけでも、減法混色のように暗くなるわけでもない。

**Q10. 正解: B（シアン C）**
加法混色における赤（R）の補色は**シアン（C）**。R + C = W（白）。
- R ↔ C（シアン）
- G ↔ M（マゼンタ）
- B ↔ Y（黄）
