## 1. 色彩体系とは

色を **数値や記号で客観的に表す** ための体系。
「この赤」「あの青」という曖昧な表現をなくし、色を正確に伝えるために使われる。

代表的な表色系：
- **マンセル表色系**（アメリカ）：日本でも広く使用
- **PCCS**（日本色研配色体系）：配色教育用
- **XYZ表色系**（CIE：国際照明委員会）：科学・工業用（→2級）
- **L\*a\*b\*表色系**（CIE）：工業測色の国際標準（→2級）

## 2. マンセル表色系

**アルバート・マンセル** が考案した表色系。
色を **色相（H）・明度（V）・彩度（C）** の3属性で表す。

### 2-1. マンセル表示法

```
H V/C
例: 5R 4/14
```

| 要素 | 表示 | 意味 |
|---|---|---|
| 色相（H） | 5R | 5番の赤（Redの5） |
| 明度（V） | 4 | 明度4（0〜10） |
| 彩度（C） | 14 | 彩度14（0〜最大値） |

無彩色は `N` で表す：
- 例: `N7` = 明度7の灰色

### 2-2. マンセルの主要色相（5色と10色）

**5主要色**：
- R（赤）・Y（黄）・G（緑）・B（青）・P（紫）

**5中間色**：
- YR（黄赤）・GY（黄緑）・BG（青緑）・PB（青紫）・RP（赤紫）

各色相は 1〜10 の数値で細分化される。
**5**（例: 5R）が各色相の代表色。

### 2-3. マンセル明度・彩度

| 明度（V） | 意味 |
|---|---|
| 0 | 完全な黒 |
| 5 | 中間のグレー |
| 10 | 完全な白 |

**彩度（C）**:
- 色相によって最大値が異なる
- 赤（R）は最大約14と高い
- 青紫（PB）は最大約12

### 2-4. マンセルの色立体（カラーソリッド）

- 縦軸 = 明度（0〜10）
- 中心 = 無彩色軸
- 水平断面 = 色相（色相環）
- 外側 = 彩度が高い

```
     白(N10)
       │
       │  (高明度・低彩度の色)
   ────┼──── 各色相が広がる
       │  (低明度・低彩度の色)
       │
     黒(N0)
```

## 3. PCCS（日本色研配色体系）

**Practical Color Co-ordinate System** の略。
1964年に **日本色彩研究所** が開発した、配色教育に特化した体系。

### 3-1. PCCSの特徴

- **ヒュートーンシステム** を採用
- 色相と明度・彩度を組み合わせた「トーン」で色を分類
- 配色設計に使いやすい

### 3-2. PCCSの色相環（24色相）

| 番号 | 色名（和） | 色名（英） | 略号 |
|---|---|---|---|
| 1 | 赤みの紫 | purplish Red | pR |
| 2 | 赤 | Red | R |
| 3 | 赤みの橙 | reddish Orange | rO |
| 4 | 橙 | Orange | O |
| 5 | 黄みの橙 | yellowish Orange | yO |
| 6 | 橙みの黄 | orangish Yellow | oY |
| 7 | 黄 | Yellow | Y |
| 8 | 緑みの黄 | greenish Yellow | gY |
| 9 | 黄みの緑 | yellowish Green | yG |
| 10 | 緑 | Green | G |
| 12 | 青みの緑 | bluish Green | bG |
| 14 | 緑みの青 | greenish Blue | gB |
| 16 | 青 | Blue | B |
| 18 | 紫みの青 | purplish Blue | pB |
| 20 | 青みの紫 | bluish Violet | bV |
| 22 | 紫 | Violet | V |
| 24 | 赤みの紫 | Red-Purple | RP |

!!! warning "試験頻出"
    2(R)・8(Y)・12(G)・18(B)・22(V) の5主要色は必ず覚える。

### 3-3. PCCSの補色（対照色相）

色相番号の差が **12** の色同士が補色関係（180°対面）。

| 色相 | 補色 |
|---|---|
| 2（赤） | 14（緑みの青）または 12（青緑） |
| 8（黄） | 20（青みの紫） |
| 18（青） | 6〜8（橙〜黄） |

### 3-4. PCCSのトーン記号と色の表示

PCCSでは色を **色相番号：トーン記号** で表す。

例：
- `v2` → ビビッドの赤（色相2・ビビッドトーン）
- `p8` → ペールの黄（色相8・ペールトーン）
- `lt12` → ライトの緑（色相12・ライトトーン）

### 3-5. ヒュートーンシステム

色を **色相（Hue）** と **トーン（Tone）** の2軸で管理するPCCS独自の体系。

```
         色相（どの色か）
            ↕
トーン（明度・彩度の組み合わせ = どんな色調か）
```

配色のときに：
- 同じ色相で、異なるトーン → **トーン配色**
- 異なる色相で、同じトーン → **色相配色**

## 4. 色相環上の色の関係

### 4-1. 同一色相（同色相）
まったく同じ色相の色同士。
例: 同じ赤の鮮やかな色と薄い色

### 4-2. 類似色相（隣接色相・準類似色相）
色相環上で隣り合う（近い）色同士。

| 関係 | PCCSの色相差 | 例 |
|---|---|---|
| 隣接色相 | 1〜2 | 赤と赤橙 |
| 類似色相 | 3〜4 | 赤と橙 |

### 4-3. 対照色相・補色色相

| 関係 | PCCSの色相差 | 例 |
|---|---|---|
| 対照色相 | 8〜10 | 赤と青 |
| 補色色相 | 12（正反対） | 赤と青緑 |

| マンセル明度 | 0（黒）〜10（白） |
| マンセル彩度 | 0（無彩色）〜約14（赤の場合） |
