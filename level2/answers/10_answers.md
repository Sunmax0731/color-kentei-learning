# 解答・解説 — 景観色彩

| 問題 | 正解 | 解説 |
|---|---|---|
| Q1 | C | 景観法は2004年施行。良好な景観の形成を促進するための法律。 |
| Q2 | C | 景観計画ではマンセル表色系を用いて外壁の色彩基準を設けることが多い。 |
| Q3 | C | アースカラー・自然素材の色（石・木・土の色）は周辺景観になじみやすい。 |
| Q4 | A | 高彩度色の面積を制限する彩度規制・色面積制限が屋外広告物規制の典型的な手法。 |
| Q5 | B | 安全色彩で「注意」は黄色。赤は防火・禁止・危険、緑は安全・救護、青は指示。 |
| Q6 | B | カラーサーベイは現地の建物・サイン・自然の色をマンセル値などで調査・記録すること。 |
| Q7 | B | 海辺には白・水色・ターコイズ系が調和しやすい。自然光・海の色彩との整合性。 |
| Q8 | A | 「防火・禁止・危険」を示す安全色彩は赤色（JIS Z 9103）。 |
| Q9 | B | 景観シミュレーションはCG等を使い、色変更が景観に与える影響を事前に確認するもの。 |
| Q10 | B | 推奨色パレットは地域の色彩風土・伝統的建築物の色を基準に策定するのが適切。 |
