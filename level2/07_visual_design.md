## 1. アパレルの色彩計画

### 1-1. ファッションの色彩シーズン

アパレル業界ではシーズンごとにカラートレンドが変化する。

| シーズン | 代表的な色調 |
|---|---|
| 春（Spring） | 明るい・淡い色（パステル）・暖色 |
| 夏（Summer） | 鮮やか・爽やか・寒色・白 |
| 秋（Autumn） | 深みのある・アーシー・暖色 |
| 冬（Winter） | 暗い・重厚・モノトーン |

### 1-2. カラーコーディネートの技法

**コーディネートの基本**:

| 方法 | 内容 |
|---|---|
| ワントーン（同一色相） | 同じ色相でトーンを変える |
| 同系色コーデ | 類似色相でまとめる |
| 補色コーデ | 補色を組み合わせてメリハリ |
| モノトーンコーデ | 白・黒・グレーでまとめる |

### 1-3. ベーシックカラーとアクセントカラー

- **ベーシックカラー**: 定番色（白・黒・グレー・ネイビー・ベージュ）
  - 多くのアイテムに使いやすく、合わせやすい
- **アクセントカラー**: 鮮やかな色・季節のトレンドカラー
  - 少量使って全体に変化をもたらす

### 1-4. トレンドカラーの流れ

1. 色彩機関（パントン等）がトレンドを発表
2. ハイブランドがコレクションで使用
3. 大手アパレルが取り入れる
4. 一般市場へ普及

## 2. インテリアの色彩計画（詳細）

### 2-1. マンセル値を使った照明計画

インテリアでは照明の色と壁・床の色の組み合わせが重要。

**壁色の選定例（マンセル表記）**:
- リビング: N7〜N8（明るいグレー）、10YR 8/2（温かいベージュ）
- 寝室: 5B 7/2（穏やかな青系）
- 子ども部屋: 10Y 8/4（明るい黄系）

### 2-2. インテリアスタイルと色彩

| スタイル | 主な色彩 | 特徴 |
|---|---|---|
| ナチュラル | ベージュ・クリーム・木の茶 | 自然素材の暖かさ |
| モダン | 白・黒・グレー・差し色 | シンプル・機能美 |
| スカンジナビアン | 白・グレー・パステル | 明るく洗練 |
| ジャパニーズ | 茶・藍・白・渋い色 | 侘び・寂び |
| インダストリアル | グレー・黒・茶（レンガ色） | 荒削り・男性的 |

### 2-3. 色の進出・後退を活かしたインテリア

| 効果 | 方法 |
|---|---|
| 天井を高く見せる | 天井を白・明るい色に |
| 部屋を広く見せる | 壁を明るい膨張色に |
| 奥行きを作る | 奥の壁を後退色（寒色・暗色）に |
| 重点を作る | 一面の壁をアクセントウォールに |

## 3. グラフィックデザインの色彩

### 3-1. 印刷・グラフィックの色彩

**CMYK（減法混色）**:
- C（シアン）+ M（マゼンタ）+ Y（イエロー）+ K（ブラック）
- 印刷物の色の表現方式

**RGB（加法混色）**:
- デジタル画面の色表現
- Webデザイン・動画制作

**カラープロファイル・色域**:
- sRGB: Web用の標準色域
- Adobe RGB: 印刷・写真向けの広色域
- CMYK変換: RGB → CMYK に変換すると色域が狭くなる（ガマット圧縮）

### 3-2. 視認性と可読性

**視認性**: 対象が見えやすいかどうか
- 明度対比が大きいほど視認性が高い
- 補色関係でも高い視認性

**可読性**: テキストが読みやすいかどうか
- 白地に黒文字が最も可読性が高い
- 彩度が高い背景色は文字を読みにくくする

### 3-3. 配色のガイドライン

```
視認性 高い: 黄色地に黒文字（警告標識の定番）
視認性 低い: 赤地に緑文字（補色だが目がちらつく）
可読性 高い: 白地に黒文字、黒地に白文字
可読性 低い: 高彩度色のみの配色
```

## 4. 景観・環境色彩

### 4-1. 景観色彩計画の目的

- 地域の自然・文化との調和
- 街並みの美観の保護・向上
- 地域アイデンティティの表現

### 4-2. 修景色彩（しゅうけいしきさい）

建物・看板・道路など、景観に影響する色彩を統一・調整すること。

**日本の事例**:
- 京都：景観条例で建物の色彩・形態を規制
- 小樽・函館：歴史的建造物の色彩保全
- 地方都市：条例で外壁色を規制している地域も多い

### 4-3. 景観の色彩のルール（一般的な基準）

- **彩度を抑える**: 高彩度の色は景観になじまないことが多い
- **自然色との調和**: 植生・地形の色に合わせる
- **歴史的文脈**: 地域の伝統色・歴史的色彩を参考にする
- **彩度の上限**: 多くの景観条例では彩度（マンセル値）の上限を設定

### 4-4. サインシステムと色彩

公共空間での案内サイン（Wayfindig）に色彩を活用：
- 色で目的地・エリアを区別
- 視認性・可読性を確保
- カラーユニバーサルデザインへの対応

## 5. バリアフリーカラーデザイン（実践）

### 5-1. 色のバリアフリーの基本原則

1. **色だけで情報を伝えない** → 形・文字・模様を組み合わせる
2. **十分なコントラストを確保** → 特に明度差
3. **赤緑の組み合わせを避ける** → D型色覚異常への配慮
4. **複数の手がかりを提供** → 点字・音声も組み合わせる

### 5-2. 推奨配色（CUD）

カラーユニバーサルデザインで推奨される色の組み合わせ：

| 使いやすい配色 | 避けたい配色 |
|---|---|
| 白×黒 | 赤×緑 |
| 黄×黒 | 赤×茶 |
| 白×青 | 緑×茶 |
| 黄×青 | 青×紫 |

### 5-3. シミュレーションツールの活用

色覚異常シミュレーターを使って、さまざまなタイプの色覚で見え方を確認する。

## 6. カラーマーケティング

### 6-1. 色と購買行動

- 赤: 衝動買いを促す・セール・SALE
- 青: 信頼感・企業のコーポレートカラーに多い
- 緑: 健康・環境・有機食品
- 黒/金: 高級感・プレミアム

### 6-2. ブランドカラー

企業・ブランドを象徴する色彩。
- 一度定着すると強力なブランド資産になる
- 競合との差別化に使われる
