## 1. マンセル表色系（詳細）

### 1-1. マンセルの補色

マンセル表色系における補色は、**混合したとき視覚的に灰色になる色同士**。

主な補色の組み合わせ：

| 色相 | 補色 |
|---|---|
| 5R（赤） | 5BG（青緑） |
| 5Y（黄） | 5PB（青紫） |
| 5G（緑） | 5RP（赤紫） |
| 5B（青） | 5YR（黄赤） |
| 5P（紫） | 5GY（黄緑） |

### 1-2. マンセルの等色相面

マンセル色立体を縦に切った断面が**等色相面（色相面）**。
- 横軸: 彩度（中心が無彩色 N、外側が高彩度）
- 縦軸: 明度（上が白、下が黒）

### 1-3. マンセルの色立体の非対称性

各色相の最高彩度が異なるため、色立体は**非対称**になる。
- 赤（R）：最高彩度が高い
- 黄緑（GY）：最高彩度が低め

### 1-4. マンセル値と実用数値

| 色 | 表示例 | 特徴 |
|---|---|---|
| 高彩度の赤 | 5R 4/14 | 最高彩度域 |
| 中程度の青 | 5B 5/8 | 一般的な青 |
| 中間グレー | N5 | 無彩色・明度5 |

## 2. PCCS（詳細）

### 2-1. PCCSの無彩色系列

PCCSでは無彩色を明度段階で表示：

| 記号 | 明度 | 意味 |
|---|---|---|
| W | 白（高明度） | White |
| Gy-9.5 | 明るいグレー | |
| Gy-8.5 | ... | |
| Gy-5 | 中間グレー | |
| Gy-1.5 | 暗いグレー | |
| Bk | 黒（低明度） | Black |

### 2-2. PCCSのトーン配置と関係

12の有彩トーンの関係（再整理）：

```
高明度    p(ペール)  ltg(ライトグレイッシュ)  lt(ライト)  b(ブライト)
         ─────────────────────────────────────────
中明度    g(グレイッシュ)  sf(ソフト)  s(ストロング)  v(ビビッド)
         ─────────────────────────────────────────
低明度    dkg(ダークグレイッシュ)  d(ダル)  dk(ダーク)  dp(ディープ)
         ←──── 低彩度 ────────────────── 高彩度 ────→
```

### 2-3. PCCSと慣用色名の対応

| 慣用色名 | PCCSトーン | 概説 |
|---|---|---|
| パステルカラー | p（ペール） | 淡く柔らかい色 |
| ビビッドカラー | v（ビビッド） | 鮮やかな色 |
| ダークカラー | dk（ダーク） | 暗い色 |
| アースカラー | d（ダル）, dkg | 土・自然の色調 |
| ネオンカラー | v（ビビッド）超高彩度 | 蛍光色 |

## 3. NCS（自然色体系）

**Natural Color System**。スウェーデンで開発された、人間の色知覚に基づく体系。

### 3-1. 基本概念

6つの基本色（エレメンタリーカラー）：

| 色 | 略号 |
|---|---|
| 白 | W |
| 黒 | S（Schwarz） |
| 黄 | Y |
| 赤 | R |
| 青 | B |
| 緑 | G |

### 3-2. NCSの表示法

```
S nncc-H
例: S 2040-Y90R
```

| 要素 | 例 | 意味 |
|---|---|---|
| S | S | 第2版を意味するプレフィックス |
| nn | 20 | 黒含有量（blackness）= 20% |
| cc | 40 | クロマ（chromaticness）= 40% |
| H | Y90R | 色相（黄90%赤10%の混合方向） |

### 3-3. NCSの三角形（色の断面）

任意の色相の色は「黒・白・純色（クロマ）」の三角形の中に位置する。

```
  白(W)
   ╱╲
  ╱  ╲
 ╱    ╲
黒(S) 純色(C)
```

## 4. オストワルト表色系

**ヴィルヘルム・オストワルト**が考案した体系。

### 4-1. 基本概念

すべての色は「純色（C）・白（W）・黒（B）」の3成分で表せる：

```
純色量（c）+ 白量（w）+ 黒量（b）= 1（100%）
```

### 4-2. 等白量・等黒量の調和

**オストワルトの調和論**：
- **等白量配色**: 白量（w）が等しい色同士は調和する
- **等黒量配色**: 黒量（b）が等しい色同士は調和する

> **試験ポイント**: オストワルトの調和は「等白量・等黒量」が鍵。

### 4-3. オストワルトの色相環

24色相（4主要色 = 黄・赤・青・緑 を基本に分割）

---

## 5. 表色系の比較まとめ

| 表色系 | 開発 | 特徴 | 用途 |
|---|---|---|---|
| マンセル | アメリカ（1905） | H V/C で直感的 | 教育・デザイン |
| PCCS | 日本（1964） | ヒュートーンシステム | 配色教育 |
| XYZ | CIE（1931） | 物理的に正確 | 科学・工業 |
| L\*a\*b\* | CIE（1976） | 知覚均等 | 工業測色 |
| NCS | スウェーデン | 知覚に基づく | 建築・インテリア |
| オストワルト | ドイツ（1920） | 3成分（純色・白・黒） | 色彩調和研究 |
