## 1. 3級配色技法の復習と深掘り

3級で学んだ技法をより実践的なレベルで理解する。

### 1-1. 色相配色の整理

| 技法 | 色相差（PCCS） | 印象 | 使いどころ |
|---|---|---|---|
| 同一色相 | 0 | まとまり・上品 | フォーマル・高級感 |
| 隣接色相 | 1〜2 | 自然・穏やか | 自然派・ナチュラル |
| 類似色相 | 3〜4 | 調和・統一 | 日常・ベーシック |
| 中差色相 | 5〜7 | 変化・個性 | カジュアル・元気 |
| 対照色相 | 8〜10 | アクティブ・メリハリ | スポーティ・POP |
| 補色色相 | 11〜12 | 強烈・インパクト | 広告・アクセント |

### 1-2. トーン配色の整理

| 技法 | トーン差 | 印象 |
|---|---|---|
| 同一トーン | 同じ | 統一感・調和 |
| 類似トーン | 隣接 | 穏やか・自然 |
| 対照トーン | 大きな差 | メリハリ・ダイナミック |

## 2. 高度な配色技法

### 2-1. ビコロール配色（Bicolor）

**2色のみ**で構成された配色。コントラストが明確。

- 特徴: シンプル・わかりやすい・インパクト
- 例: 赤×白、黒×金
- 注意: バランスと色の関係（補色か類似かで印象が変わる）

### 2-2. トリコロール配色（Tricolor）

**3色**で構成された配色。フランス国旗（青・白・赤）が典型例。

- 特徴: 国旗配色として有名・鮮やかで力強い
- 3色の色相差が大きいと強いコントラスト

### 2-3. ドミナントカラー配色（Dominant Color）

**特定の色相を支配的に使う**配色。

- 特徴: 統一感・色相のまとまり
- 例: 全体を青系でまとめ、他の色相を控えめにする

### 2-4. ドミナントトーン配色（Dominant Tone）

**特定のトーンを支配的に使う**配色。色相が異なっても統一感が生まれる。

- 特徴: 雰囲気の統一・配色の方向性が明確
- 例: ペールトーンのみで多色配色 → 柔らかく上品

### 2-5. トーナル配色（Tonal）

**中彩度のトーン（ダル・ソフト・グレイッシュなど）**でまとめた配色。

- 特徴: 落ち着き・渋み・アーシー
- 例: アースカラーのインテリア

### 2-6. カマイユ配色（Camaïeu）

**同一または非常に近い色相**で、トーンの差もほとんどない配色。

- 特徴: 色の差がほとんどなく一体感が強い・モノトーンに近い
- 「camaïeu」＝フランス語で単色の浮き彫りを意味する

### 2-7. フォカマイユ配色（Faux Camaïeu）

カマイユよりやや色相差・トーン差がある配色（「偽のカマイユ」）。

- 特徴: 色の差はあるが、まとまりがある・微妙な変化
- カマイユとビコロールの中間的な印象

### 2-8. グラデーション配色（Gradation）

色相・明度・彩度が段階的に変化する配色。

- 色相グラデーション: 虹の色のような連続変化
- 明度グラデーション: 白〜グレー〜黒 の段階変化
- 彩度グラデーション: 鮮やか → くすみ の段階変化

### 2-9. セパレーション配色（Separation）

対立する2色の間に**白・黒・グレー（または細線）**を挿入して色のぶつかりを和らげる。

- 使用場面: 補色配色で目がちらつくとき、対照色相の配色を落ち着かせたいとき

### 2-10. リピティション配色（Repetition）

同じ色の組み合わせを**繰り返す**配色。

- 特徴: リズム感・秩序感・模様的な印象
- 例: ストライプ・チェック・ボーダー

### 2-11. アクセント配色（Accent）

全体の配色に**少量の目立つ色**を加える。

- 面積比: 全体の5〜10%程度
- 目的: 単調さの解消・視線の誘導

## 3. 配色のバランスと評価

### 3-1. 配色の評価基準

良い配色とは：
1. **目的・コンセプトに合っている**
2. **ターゲットに合っている**
3. **機能的（見やすい・識別しやすい）**
4. **美的に調和している**

### 3-2. 色の対比と同化を配色に活かす

| 効果 | 配色の方向性 |
|---|---|
| 明度対比を活かす | 背景と前景の明度差を大きくして見やすくする |
| 彩度対比を活かす | 低彩度の背景にアクセントカラーを置く |
| 面積対比を考慮 | 大面積の色は実際より鮮やかに見える → 少しくすんだ色を選ぶ |

### 3-3. 色数と複雑さ

| 色数 | 印象 |
|---|---|
| 2色（ビコロール） | シンプル・明快 |
| 3色（トリコロール） | バランス・変化 |
| 4〜5色 | 豊か・賑やか |
| 6色以上 | 複雑・にぎやか（まとめにくい） |

## 4. 配色技法の選び方

### 用途別の配色選択

| 用途 | 推奨配色 |
|---|---|
| フォーマル・高級 | 同一色相・トーナル・モノトーン |
| カジュアル・ポップ | 対照色相・高彩度 |
| ナチュラル・エコ | 類似色相・アースカラー（トーナル） |
| 落ち着き・信頼 | 寒色系・低彩度・ドミナントカラー |
| 目立つ・広告 | 補色・高彩度・ビコロール |
| 女性らしさ | ペールトーン・ピンク系・カマイユ |
