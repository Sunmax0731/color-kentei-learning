# 3級 — 色彩心理

> **期間**: Week 5（4/13〜4/19）｜目安時間: 10時間

## 目標

- 暖色・寒色・中性色の違いを説明できる
- 色の感情効果（重さ・硬さ・大小・遠近）を答えられる
- 色の連想と象徴を理解できる
- 色の嗜好傾向を把握できる

---

## 1. 暖色・寒色・中性色

### 暖色（Warm Colors）
**温かみ・暑さ・熱さ** を感じさせる色。

- 色相: 赤〜赤橙〜橙〜黄橙 あたり
- 例: 赤、橙、黄橙
- 印象: 活動的・興奮・情熱・暖かい

### 寒色（Cool Colors）
**冷たさ・涼しさ・落ち着き** を感じさせる色。

- 色相: 青緑〜青〜青紫 あたり
- 例: 青、青緑、青紫
- 印象: 落ち着き・冷静・清涼感

### 中性色（Neutral Colors）
暖かくも冷たくも感じられない色。

- 色相: 黄・黄緑・緑・紫 あたり
- 黄緑・緑・紫などは文脈によって暖かくも冷たくも感じられる

```
暖色 ←──────── 中性色 ────────→ 寒色
（赤・橙）（黄緑・緑・紫）（青緑・青）
```

> **試験頻出**: 「最も暖かく感じる色 = 橙系」「最も冷たく感じる色 = 青系」

## 2. 色の感情効果

### 2-1. 色の重さ感

| 感覚 | 色の特徴 | 例 |
|---|---|---|
| 重く感じる | 明度が **低い** 色 | 黒・濃い紺・ダークブラウン |
| 軽く感じる | 明度が **高い** 色 | 白・淡いクリーム・ペールブルー |

> **ポイント**: 重さ感は主に **明度** に関係する（色相よりも明度が大きく影響）。

### 2-2. 色の硬さ感・柔らかさ感

| 感覚 | 色の特徴 |
|---|---|
| 硬く感じる | 明度が低い・彩度が高い |
| 柔らかく感じる | 明度が高い・彩度が低い |

### 2-3. 色の大小感（膨張色・収縮色）

| 感覚 | 色の特徴 | 例 |
|---|---|---|
| 大きく見える（膨張色） | 明度が高い・暖色系 | 白・黄・赤 |
| 小さく見える（収縮色） | 明度が低い・寒色系 | 黒・紺・濃い青 |

> **身近な例**: スポーツ選手が白いユニフォームを着ると大きく、黒いユニフォームだと引き締まって見える。

### 2-4. 色の遠近感（進出色・後退色）

| 感覚 | 色の特徴 | 例 |
|---|---|---|
| 近く・前に見える（進出色） | 暖色系・明度が高い | 赤・橙・黄 |
| 遠く・奥に見える（後退色） | 寒色系・明度が低い | 青・青紫 |

> **身近な例**: 交通標識の「注意」は赤・黄（進出色）を使い目立たせる。

### 2-5. 色の興奮・沈静効果

| 効果 | 色の特徴 |
|---|---|
| 興奮・活性化 | 暖色系・高彩度（赤・橙） |
| 沈静・落ち着き | 寒色系・低彩度（青・青緑） |

## 3. 色の連想

ある色を見たとき、関連するものやイメージが思い浮かぶこと。

### 具体的連想（実際の物を連想）

| 色 | 連想するもの |
|---|---|
| 赤 | 血・りんご・信号・ポスト・消防車 |
| 黄 | バナナ・ひまわり・信号 |
| 緑 | 草・木・自然・新鮮さ |
| 青 | 空・海・水・氷 |
| 白 | 雪・白衣・白紙 |
| 黒 | 夜・土・石炭 |

### 抽象的連想（概念・感情を連想）

| 色 | 連想する概念 |
|---|---|
| 赤 | 情熱・危険・愛・活力 |
| 橙 | 活気・陽気・温かさ |
| 黄 | 希望・元気・注意 |
| 緑 | 安全・平和・自然・新鮮 |
| 青 | 冷静・知性・信頼・誠実 |
| 紫 | 高貴・神秘・高級感 |
| 白 | 清潔・純粋・無垢 |
| 黒 | 威厳・高級感・死・喪 |
| 茶 | 自然・土・安定感 |
| ピンク | 可愛さ・愛らしさ・女性らしさ |

## 4. 色の象徴

色には文化・慣習によって決まった**象徴的意味**がある。

### 日本での色の象徴

| 色 | 象徴 |
|---|---|
| 赤 | 魔除け・祝い・生命 |
| 白 | 清潔・神聖・喪（白装束） |
| 黒 | 喪・格式・正式（礼服） |
| 紺 | 信頼・誠実 |
| 金 | 富・名誉 |

### 交通信号の色の意味

| 色 | 意味 |
|---|---|
| 赤 | 停止・危険 |
| 黄 | 注意 |
| 青（緑） | 進行・安全 |

### 安全色彩（日本工業規格 JIS Z 9101）

| 色 | 意味 |
|---|---|
| 赤 | 防火・禁止・停止 |
| 黄赤（橙） | 危険 |
| 黄 | 注意 |
| 緑 | 安全・救護・進行 |
| 青 | 指示 |
| 赤紫 | 放射能 |
| 白 | 通路・整理 |
| 黒 | 補助色 |

## 5. 色の嗜好

**色の好みには個人差・文化差・年齢差**がある。

### 日本での一般的傾向

- 男性に人気: 青系・紺系
- 女性に人気: ピンク・赤・紫
- 子どもに人気: 赤・黄・橙などの鮮やかな暖色
- 年齢とともに: 高彩度から低彩度へ、暖色から寒色・無彩色への傾向

> **試験ポイント**: 嗜好には個人差・文化差があり、普遍的な正解はない。ただし傾向として「青が世界的に好まれやすい」とされる。

## 6. 色の心理的効果まとめ表

| 効果 | 主に影響する属性 | 例 |
|---|---|---|
| 重さ感 | 明度 | 暗い色 = 重い |
| 硬さ感 | 明度・彩度 | 低明度・高彩度 = 硬い |
| 大きさ感 | 明度・色相 | 高明度・暖色 = 大きい |
| 遠近感 | 色相・明度 | 暖色 = 近い/寒色 = 遠い |
| 興奮・沈静 | 色相・彩度 | 赤高彩度 = 興奮 |
