# 解答・解説 — 色のユニバーサルデザイン

| 問題 | 正解 | 解説 |
|---|---|---|
| Q1 | B | 日本人男性の約5%（女性は約0.2%）がP型またはD型色覚。試験頻出の数値。 |
| Q2 | B | P型・D型では赤と緑が最も混同しやすい。これが「赤緑色覚異常」と呼ばれる理由。 |
| Q3 | C | CUDの基本は「色だけで情報を伝えない」こと。色＋形・文字・パターンを組み合わせる。 |
| Q4 | C | 加齢による水晶体の黄変で短波長（青・紫）が透過しにくくなる。青が見えにくくなる。 |
| Q5 | C | グラフでは色＋線の種類・パターン（実線・破線・ハッチング）を組み合わせるのが適切。 |
| Q6 | B | D型はM錐体（緑を感じる錐体）の機能が失われている。P型はL錐体（赤）。 |
| Q7 | B | CUDカラーパレットは公共の情報グラフィック（路線図・グラフ・サインなど）で活用する。 |
| Q8 | C | ウェブではリンクに下線などを加え、色以外の手がかりを与えるのが適切。 |
| Q9 | C | 路線図は色に加えて路線番号・アルファベット記号を付記するのがCUDの実践。 |
| Q10 | C | 個人の日記の文字色は私的な範囲で、CUDが特に重要な公共性の高い領域ではない。 |
