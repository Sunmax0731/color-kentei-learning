## 1. 光とは何か

色の正体は **光** である。光は電磁波の一種で、私たちが「色」として認識できるのは **可視光線** の波長域に限られる。

```
電磁波の種類（波長が短い順）
ガンマ線 → X線 → 紫外線 → 【可視光線】 → 赤外線 → 電波
```

### 可視光線の波長域

**約 380nm（ナノメートル）〜 780nm**

| 色 | おおよその波長域 |
|---|---|
| 紫（バイオレット） | 380〜430nm |
| 青（ブルー） | 430〜480nm |
| 青緑（シアン寄り） | 480〜500nm |
| 緑（グリーン） | 500〜560nm |
| 黄（イエロー） | 560〜590nm |
| 橙（オレンジ） | 590〜620nm |
| 赤（レッド） | 620〜780nm |

!!! tip "ポイント"
    波長が短いほど紫寄り、長いほど赤寄り。
!!! warning "試験頻出 :star:"
    可視光線の範囲「380〜780nm（約400〜700nm）」は必ず覚える。

## 2. 色が見えるしくみ

色が見えるためには以下の3要素が必要：

```
光源（光） → 物体（反射・吸収・透過） → 眼（視覚）
```

1. **光源** が光を放つ
2. **物体** が特定の波長を反射し、他の波長を吸収する
3. **眼の網膜** にある錐体細胞（L・M・S）が光を感知して脳が色と認識する

---

## 3. 色の分類

### 3-1. 光源色（自発光色）
自ら光を発するものの色。
- 例: 太陽光、ろうそく、電球、モニター画面

### 3-2. 物体色
光が物体に当たり、特定の波長が反射されることで見える色。
- **表面色（反射色）**: 不透明な物体の表面の色（白い紙、塗料など）
- **透過色**: 透明・半透明な物体を透過した光の色（ステンドグラス、サングラスのレンズなど）
- **空間色**: 空間に色が広がって見える色（青空など）
- **鏡面色（光沢色）**: 金属など光沢のある表面から反射される色

---

## 4. 光源の種類と特徴

| 光源 | 特徴 | 色温度（目安） |
|---|---|---|
| 晴天昼光（青空） | 青みがかった白色光 | 約6000〜7000K |
| 曇り空 | 均一な拡散光 | 約6500K |
| 白熱電球 | 赤みがかった暖色の光 | 約2700〜3000K |
| 蛍光灯（昼光色） | 青みがかった白色 | 約6500K |
| 蛍光灯（電球色） | 温かみのある白色 | 約2700〜3000K |
| LED | 種類により様々 | 2700〜6500K |

**色温度（Color Temperature）**
光源の色みを表す単位（K: ケルビン）。
- 低い（2700K）→ 暖色系（赤・橙みがかった白）
- 高い（6500K）→ 寒色系（青みがかった白）

---

## 5. 演色性（えんしょくせい）

**演色性** とは、光源が物体の色をどれだけ忠実に見せることができるかの指標。

- **Ra（平均演色評価数）**: 0〜100の数値で表す。100に近いほど高演色性
- **基準光源**: 太陽光（昼光）が Ra = 100

| 光源 | 演色性 Ra（目安） |
|---|---|
| 太陽光 | 100 |
| 白熱電球 | 100に近い |
| 三波長蛍光灯 | 80〜85 |
| 一般蛍光灯 | 60〜70 |

!!! info "用途との関係"
    美術館・絵画・宝石・衣料品など色の見え方が重要な場所では **高演色性** の光源が求められる。

---

## 6. メタメリズム（条件等色）

**メタメリズム** とは、2つの色が **特定の光源の下では同じ色に見えるが、別の光源の下では異なる色に見える** 現象のこと。

- 英語: Metamerism（メタメリズム）
- 日本語: 条件等色（じょうけんとうしょく）

```
例:
昼光（6500K）の下で → AとBが同じ灰色に見える
電球（2700K）の下で → AはわずかにオレンジがかりBはやや青みがかる
```

**起こる原因**: 2つの物体の分光反射率が異なるが、特定の光源のスペクトルとの掛け合わせで同じ三刺激値になるため。

!!! warning "試験頻出 :star:"
    メタメリズムは「同じ光源で同色に見えるが、光源が変わると異なる色に見える」現象。

---

## 7. 分光反射率（参考）

物体が各波長の光をどの割合で反射するかを示したグラフ。
- 白い物体：すべての波長をほぼ均等に反射（高い反射率）
- 黒い物体：すべての波長をほぼ吸収（低い反射率）
- 赤い物体：長波長（赤）を多く反射し、短波長（青）を吸収

## 参考：色の見え方に影響する要因まとめ

| 要因 | 影響 |
|---|---|
| 光源の種類 | 同じ物体でも色が違って見える |
| 光源の演色性 | 低演色性の光源では色が不自然に見える |
| 背景・周囲の色 | 対比効果により色が変わって見える（→Week5） |
| 観察者の順応 | 長時間同じ色を見ていると感覚が変わる（→2級） |
