# 解答・解説 — 配色イメージ

| 問題 | 正解 | 解説 |
|---|---|---|
| Q1 | B | イメージスケールの縦軸はSOFT（柔らかい）/ HARD（硬い）。横軸はWARM / COOL。 |
| Q2 | C | ロマンティックなイメージにはpale（p）やlight（lt）など高明度・低彩度のトーンが適する。 |
| Q3 | D | 「シック・モダン・クール」はHARD × COOL（右下）の象限。 |
| Q4 | C | 秋は紅葉・枯れ草のイメージ。中〜低明度・アースカラー（オレンジ・茶・カーキ）が特徴。 |
| Q5 | B | カジュアルは中〜高彩度・多色使い・暖色〜中性色が特徴。明るく活発な印象。 |
| Q6 | B | ビジネスパーソン向けにはモダン・フォーマル（低彩度・無彩色系）が適する。 |
| Q7 | B | ゴージャスにはvivid（v）＋deep（dp）の組み合わせが豪華な印象を生む。 |
| Q8 | A | 「やさしい・ナチュラル」はSOFT × WARM（左上）の象限。 |
| Q9 | B | 春は高明度・低〜中彩度・暖色（桜ピンク・若草色など）が特徴。 |
| Q10 | C | vivid（v）は鮮やか・元気・派手なイメージ。高彩度が特徴のトーン。 |
