# 3級 — 混色

> **期間**: Week 4（4/6〜4/12）｜目安時間: 10時間

---

## 目標

- 加法混色・減法混色・中間混色の違いを説明できる
- 各混色の原色を覚えられる
- 混色後の色の変化を予測できる
- 補色混色の結果（白・黒・灰）を答えられる

---

## 1. 混色の種類

色を混ぜると新しい色が生まれる。混ぜ方には大きく3種類ある。

| 種類 | 混ぜるもの | 混ぜるほど |
|---|---|---|
| 加法混色 | 光 | 明るくなる |
| 減法混色 | 色料（絵具・インク） | 暗くなる |
| 中間混色 | 光の回転・並置 | 中間の明るさ |

---

## 2. 加法混色（光の混色）

**光を重ねることで色を作る** 混色。光を重ねるほど明るくなる。

### 三原色

**R（赤）・G（緑）・B（青）**

```
R + G + B = 白（白色光）
```

| 混合 | 結果 |
|---|---|
| R（赤） + G（緑） | Y（黄） |
| G（緑） + B（青） | C（シアン） |
| B（青） + R（赤） | M（マゼンタ） |
| R + G + B | W（白） |
| なし | K（黒 = 光がない状態） |

> **ポイント**: 加法混色では、混ぜるほど **明るく** なり、全部混ぜると **白** になる。

### 身近な例

- テレビ・スマートフォンの画面
- 舞台照明・スポットライト
- プロジェクターの映像

### 補色の関係（加法混色）

加法混色における補色は、2色を足すと白になる色同士。

| 色 | 補色 |
|---|---|
| R（赤） | C（シアン） |
| G（緑） | M（マゼンタ） |
| B（青） | Y（黄） |

---

## 3. 減法混色（色料の混色）

**色料（絵具・インクなど）を混ぜることで色を作る** 混色。色料は光を吸収するため、混ぜるほど暗くなる（光が減る）。

### 三原色

**C（シアン）・M（マゼンタ）・Y（イエロー）**

```
C + M + Y = 黒（理論上）/ 暗い茶（実際）
```

| 混合 | 結果 |
|---|---|
| C（シアン） + M（マゼンタ） | B（青紫） |
| M（マゼンタ） + Y（イエロー） | R（赤） |
| Y（イエロー） + C（シアン） | G（緑） |
| C + M + Y | K（黒 ※理論上） |

> **ポイント**: 減法混色では、混ぜるほど **暗く** なり、全部混ぜると **黒** になる。

### 身近な例

- 絵具・水彩・油絵
- 印刷（CMYK：シアン・マゼンタ・イエロー・ブラック）
- インクジェットプリンター

### 補色の関係（減法混色）

減法混色における補色は、2色を足すと黒になる色同士（CMYそれぞれの関係）。

| 色 | 補色 |
|---|---|
| C（シアン） | R（赤） |
| M（マゼンタ） | G（緑） |
| Y（イエロー） | B（青） |

---

## 4. 加法混色と減法混色の比較まとめ

| 項目 | 加法混色 | 減法混色 |
|---|---|---|
| 混ぜるもの | 光 | 色料（絵具・インク） |
| 三原色 | RGB（赤・緑・青） | CMY（シアン・マゼンタ・イエロー） |
| 全部混ぜると | 白 | 黒 |
| 混ぜるほど | 明るくなる | 暗くなる |
| 用途 | 画面・照明 | 印刷・絵具 |

---

## 5. 中間混色

**中間混色** は、光量は増減せず、色が「中間的に」見える混色。加法混色とも減法混色とも異なる。

### 5-1. 回転混色（回転体混色）

複数の色を塗ったコマ（円板）を高速回転させると、色が混合して見える。

- 例: 赤と白の円板を高速回転 → ピンクに見える

明度は両色の **中間**（平均）になる。

### 5-2. 並置混色（空間混色）

小さな異なる色の点（ドット）を並べると、離れて見ると色が混合して見える。

- 例: 新聞・雑誌の印刷（網点）、モザイクアート、ゴッホなどの点描画
- 例: テレビ画面のRGB画素

> **ポイント**: 中間混色の明度は、混色する各色の **平均値** になる（加法混色のように明るくはなならい）。

---

## 6. 補色混色の結果まとめ

補色同士を混ぜると：

| 混色の種類 | 結果 |
|---|---|
| 加法混色（光の補色） | 白 |
| 減法混色（色料の補色） | 黒（または暗い濁色） |
| 中間混色（回転・並置） | 灰色 |

---

## 7. よく出る混色の計算例

### 加法混色の計算例

```
R + G = Y（黄）
Y + B = W（白）  ← R + G + B = W
```

### 減法混色の計算例

```
C + Y = G（緑）
C + M = B（青紫）
M + Y = R（赤）
```
