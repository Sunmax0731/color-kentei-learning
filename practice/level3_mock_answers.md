## 重要ポイントの再確認

### 光と色
- 可視光線: **380〜780nm**
- 演色評価数 Ra: 100 = 太陽光、高いほど忠実
- メタメリズム: 光源が変わると色が違って見える
- 色温度: 低い（2700K）= 暖色、高い（6500K）= 寒色

### 色の三属性
- 三属性: **色相・明度・彩度**
- 純色 + 白 = **明清色** / 純色 + 黒 = **暗清色** / 純色 + グレー = **濁色**
- ビビッド（v）= 最高彩度 / ペール（p）= 最高明度・最低彩度

### 色彩体系
- マンセル: `H V/C`（例: 5R 4/14）/ 無彩色 = N
- PCCS: 24色相 / 2=R, 8=Y, 12=G, 18=B, 22=V
- 補色色相差: **12**

### 混色
- 加法混色（光）: RGB → 全部で白 / R+G=Y, G+B=C, B+R=M
- 減法混色（色料）: CMY → 全部で黒 / C+M=B, M+Y=R, Y+C=G
- 中間混色: 明度は平均値

### 色彩心理
- 重さ感: **明度**が低い = 重い
- 進出色: **暖色・高明度**
- 安全色: 赤=防火禁止、橙=危険、黄=注意、緑=安全

### 配色・応用
- 補色配色: 色相差12 / 最強コントラスト
- セパレーション: 2色の間に白・黒を挟む
- ベースカラー: **70%** / アクセント: **5%**
- パーソナルカラー: ブルーベース = サマー・ウィンター
