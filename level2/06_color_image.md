---

## 1. 配色イメージとは

**配色イメージ** とは、複数の色を組み合わせたときに生まれる全体的な印象・感覚のことです。同じ色でも組み合わせ方によってまったく異なるイメージを生み出します。

!!! tip "ポイント"
    単色のイメージ（色の感情効果）と、配色全体のイメージは区別して考える。配色のイメージはトーンの組み合わせに大きく依存する。

---

## 2. イメージスケール

**イメージスケール** は、色・配色のイメージをキーワード（形容詞）と対応させて可視化したツールです。日本色研事業株式会社が開発したPCCSベースのイメージスケールが代表的です。

### 2軸の構造

| 軸 | 方向 | 説明 |
|---|---|---|
| 縦軸 | SOFT ↔ HARD | 柔らかい・軽い ↔ 硬い・重い |
| 横軸 | WARM ↔ COOL | 暖かい・動的 ↔ 冷たい・静的 |

!!! warning "試験頻出"
    イメージスケールの縦軸（SOFT/HARD）と横軸（WARM/COOL）の意味、各象限のキーワードは必ず覚える。

### 4象限のイメージ

| 象限 | キーワード例 |
|---|---|
| SOFT × WARM（左上） | かわいい・ロマンティック・やさしい・ナチュラル |
| SOFT × COOL（右上） | 清潔・さわやか・エレガント・上品 |
| HARD × WARM（左下） | 元気・にぎやか・カジュアル・ダイナミック |
| HARD × COOL（右下） | シック・モダン・クール・フォーマル |

---

## 3. トーンとイメージの関係

配色イメージはトーン（明度＋彩度のまとまり）によって大きく左右されます。

| トーン | イメージ |
|---|---|
| vivid（v） | 鮮やか・元気・派手 |
| bright（b） | 明るい・さわやか・若々しい |
| pale（p） | やさしい・ロマンティック・かわいい |
| light（lt） | ライト・ナチュラル・やわらか |
| soft（sf） | 穏やか・落ち着いた・上品 |
| dull（d） | くすんだ・大人っぽい・洗練 |
| deep（dp） | 深み・伝統的・重厚 |
| dark（dk） | 男性的・ハード・重い |
| light grayish（ltg） | 上品・繊細・おとなしい |
| grayish（g） | 地味・渋い・落ち着いた |
| dark grayish（dkg） | 硬い・重い・頑固 |

## 4. 代表的な配色イメージとキーワード

### ナチュラル

- **色の特徴**: 低〜中彩度の暖色系・アースカラー・グリーン
- **トーン**: lt・sf・d・ltg
- **キーワード**: 自然・素朴・温もり・オーガニック

### ロマンティック

- **色の特徴**: 高明度の暖色系・ピンク・ラベンダー
- **トーン**: p・lt・b
- **キーワード**: かわいい・夢見がち・甘い・女性的

### エレガント

- **色の特徴**: 中〜高明度・低〜中彩度・紫系・グレイッシュ
- **トーン**: sf・ltg・lt
- **キーワード**: 上品・優雅・洗練・フェミニン

### カジュアル

- **色の特徴**: 中〜高彩度・多色使い・暖色〜中性色
- **トーン**: b・v・lt
- **キーワード**: 気軽・活発・にぎやか・若々しい

### モダン

- **色の特徴**: 低彩度・無彩色＋アクセントカラー・寒色系
- **トーン**: g・dkg＋v や b のアクセント
- **キーワード**: 都会的・洗練・スタイリッシュ・知的

### ゴージャス

- **色の特徴**: 高彩度・金・紫・深紅
- **トーン**: v・dp
- **キーワード**: 豪華・華やか・贅沢・格調

### トラディショナル

- **色の特徴**: 深みのある暖色系・ダークカラー
- **トーン**: dp・dk・dkg
- **キーワード**: 伝統・格式・重厚・安定

## 5. 目的別イメージ配色の選び方

### ターゲットとイメージ配色

| ターゲット | 適するイメージ配色 |
|---|---|
| 子ども・ファミリー | カジュアル・ポップ・フレッシュ |
| 若い女性 | ロマンティック・フェミニン・エレガント |
| ビジネスパーソン | モダン・フォーマル・シック |
| 高齢者 | トラディショナル・ナチュラル |
| 男性全般 | ダンディ・クール・ダイナミック |

### 季節とイメージ配色

| 季節 | 配色イメージの傾向 |
|---|---|
| 春 | ロマンティック・ナチュラル・フレッシュ（高明度・低〜中彩度・暖色） |
| 夏 | クール・フレッシュ・ダイナミック（高明度・高彩度・寒色〜白） |
| 秋 | ナチュラル・トラディショナル（中〜低明度・アースカラー） |
| 冬 | フォーマル・モダン・ゴージャス（低明度・無彩色・深色） |

---

## 6. 配色イメージと文化的背景

色のイメージは文化・地域によって異なる場合があります。

!!! warning "試験ポイント"
    同じ色でも文化によってイメージが異なる。例えば「白」は日本では喪のイメージ（葬儀）と清潔のイメージが共存するが、西洋では主に清潔・純潔のイメージ。
