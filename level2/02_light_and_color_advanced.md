## 1. 分光分布（Spectral Power Distribution）

光源が各波長の光をどの強さで放出しているかを示したグラフ。

```
強度
│    ／＼
│   /   \
│  /     ＼
│ /        ＼
│/           ＼
└─────────────── 波長(nm)
  380    780
```

### 1-1. 代表的な光源の分光分布

| 光源 | 分光分布の特徴 |
|---|---|
| 太陽光（D65） | 全波長域にわたって比較的均等に分布 |
| 白熱電球 | 長波長（赤〜橙）側が強い右肩上がりの連続スペクトル |
| 蛍光灯 | 連続スペクトルに不連続な輝線（ピーク）が重なる |
| LED | 青のピーク＋蛍光体による広帯域放射 |
| ナトリウムランプ | 特定波長のみ（589nm付近）の輝線スペクトル |

### 1-2. 標準光源（CIE）

| 記号 | 説明 |
|---|---|
| A光源 | 白熱電球に相当（色温度 2856K） |
| D65光源 | 昼光（6500K）：最も一般的な標準光源 |
| D50光源 | 昼光（5000K）：印刷・カラーマネジメントで使用 |
| C光源 | 平均昼光（旧標準） |
| F光源 | 蛍光灯（F2, F7, F11など種類がある） |

## 2. 等色関数と三刺激値

### 2-1. 等色関数（Color Matching Function）

人間の眼の錐体細胞（L・M・S）の分光感度特性を標準化したもの。

**CIE 1931標準観察者の等色関数**: x̄(λ)・ȳ(λ)・z̄(λ)

```
感度
│     ȳ
│    ／＼
│   /   ＼
│  / x̄   ＼
│ /   \   z̄\
│/     \  /  \
└──────────────── 波長(nm)
 380       780
```

### 2-2. 三刺激値（Tristimulus Values）

光源の分光分布と等色関数を掛け合わせて積分した値。

```
X = ∫ P(λ) · x̄(λ) dλ
Y = ∫ P(λ) · ȳ(λ) dλ
Z = ∫ P(λ) · z̄(λ) dλ
```

- **Y** は明度（輝度）に相当
- X・Y・Z の3つで色を数値的に表現できる

## 3. XYZ表色系（CIE XYZ）

CIE（国際照明委員会）が1931年に制定した表色系。

### 3-1. xy色度座標（色度図）

三刺激値 X・Y・Z から、明度成分を除いた「色み」のみを表す座標。

```
x = X / (X + Y + Z)
y = Y / (X + Y + Z)
z = Z / (X + Y + Z)  ※ z = 1 - x - y
```

**xy色度図（馬蹄形）**:
- 外周（スペクトル軌跡）: 各波長の純粋な光の位置
- 内側: 実際の物体色・光源色の範囲
- 中心付近（白色点）: 白・グレーの位置（D65: x≈0.313, y≈0.329）

### 3-2. 色域（ガマット）

特定のデバイス（ディスプレイ・プリンター）が表現できる色の範囲。
xy色度図上の三角形で表すことが多い。

## 4. L\*a\*b\*表色系（CIELAB）

1976年にCIEが制定。**知覚的に均等（Perceptually Uniform）** な色空間。

```
L* = 明度（0 = 黒、100 = 白）
a* = 赤緑方向（+a* = 赤、-a* = 緑）
b* = 黄青方向（+b* = 黄、-b* = 青）
```

### 4-1. 特徴

- XYZ表色系より人間の知覚に近い（等間隔に近い）
- **色差（ΔE）** の計算に使用される
- 工業測色・カラーマネジメントの国際標準

### 4-2. 色差（ΔE）

2色の違いの大きさを数値で表す。

```
ΔE = √[(ΔL*)² + (Δa*)² + (Δb*)²]
```

- ΔE < 1：人間がほとんど区別できない差
- ΔE = 1〜3：わずかに感じる差
- ΔE > 5：明確に異なると感じる差

---

## 5. L\*C\*h\*表色系（CIELCH）

L\*a\*b\*を極座標で表したもの。人間の感覚に合わせた表示方法。

```
L* = 明度
C* = 彩度（クロマ / 中心からの距離）
h  = 色相角（0°〜360°）
```

- h = 0°: 赤（+a*方向）
- h = 90°: 黄（+b*方向）
- h = 180°: 緑（-a*方向）
- h = 270°: 青（-b*方向）

---

## 6. 測色（Color Measurement）

### 6-1. 分光光度計（スペクトロフォトメーター）

物体の**分光反射率**を測定する機器。
測定した分光反射率と標準光源の分光分布から三刺激値を計算する。

### 6-2. 色彩計（カラーメーター）

三刺激値を直接測定する機器。分光光度計より簡易。

### 6-3. 測色条件（CIE推奨）

| 条件 | 内容 |
|---|---|
| 0°/45° または 45°/0° | 照明・受光角度の標準条件 |
| 拡散照明 d/0° | 積分球使用 |

---

## 7. 演色評価数（Ra）の詳細

### 7-1. 演色評価の基準光源

- 色温度 5000K未満: A光源が基準
- 色温度 5000K以上: D65（またはC）光源が基準

### 7-2. 8種のテストカラーサンプル

CIE が定めた8色の標準試験色に対して、基準光源との色ずれを評価する。
それぞれの特殊演色評価数 Ri の平均が Ra。

### 7-3. 高演色性LEDの表記

Ra ≥ 90 を「高演色性」と呼ぶことが多い。
美術館・精密印刷・医療など色の正確な認識が重要な場面では Ra ≥ 95 が求められることも。
